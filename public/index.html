Lista de Web Services y entidades

Web Services

Método Descripción Tipo Petición In Out
/user/login Método para iniciar sesión en la aplicación POST LoginRequest LoginResponse
/user/register Método para registrar un nuevo usuario POST RegisterRequest LoginResponse
/user/registerfacebook Método para registrar usuario con facebook POST RegisterFacebookRequest LoginResponse
/feed/all Método para traer todos los feeds de los amigos POST FeedRequest FeedResponse

Entidades

  1. User
    • id: int
    • name: string
    • email: string
    • password: string
    • facebookId: string
    • imageUrl: string
  2. Quest
    • id: int
    • latitude: double
    • longitude: double
    • name: string
    • address: string
    • hint: string
    • brief: text
    • difficulty: int
  3. LoginRequest
    • email: string - correo electrónico del usuario
    • password: string - contraseña con codificación MD5
  4. Loginresponse
    • user: User - entidad de usuario
  5. RegisterRequest
    • name: string - nombre completo del usuario
    • email: string - correo electrónico del usuario
    • password: string - contraseña con codificación MD5
  6. RegisterFacebookRequest
    • name: string - nombre completo del usuario
    • email: string - correo electrónico del usuario
    • password: string - contraseña con codificación MD5
  7. FeedRequest
    • userId: int - id del usuario que realiza la petición
    • feedUserid: int - id del usuario a consultar (null si es all)
  8. FeedResponse
    • feedList: Array - listado de feeds
  9. FeedEntity
    • userId: int - id del usuario a quien pertenece el feed
    • userName: string - nombre del usuario del feed
    • userPictureUrl: string - url de la imagen del usuario
    • dateFeed: string - fecha de cuando fué ganado (fecha epoch)
    • questName: string - nombre del quest
    • questId: int - identificador del quest
